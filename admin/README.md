# Dark Deco Admin

The Dark Deco Admin panel feartures a place for administrators to login to the site, where they can upload new products, view available products & process orders.

## Project Features

-HTML
-CSS
-React
-Tailwind
-Toastify

## Running The Project

npm install

npm run dev
